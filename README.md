# 💰 Expense Splitter (Java OOP Project)

A console-based Expense Splitter application built using Java and Object-Oriented Programming.

This project helps users split expenses like Splitwise and track who owes whom.

---

## 🚀 Features

- Add Users
- Add Expenses
  - Equal Split
  - Exact Split
  - Percentage Split
- Track Balances
- Show Settlement (who pays whom)
- Menu-driven console interface

---

## 🧠 OOP Concepts Used

- Encapsulation (private variables)
- Abstraction (separate classes)
- Polymorphism (different split types)
- Composition (Group → Users → Expenses)

---
